Learn More & Project Progress

Since the initial project concept in October 2023, significant progress has been made, including several major changes:

  • Progress Made:
    • Optimized Laser Tracking for improved precision in user interactions.
    • New User Interaction Mode allowing interactions with uploaded images.
    • Senior Projects Website to showcase our work and engage with a wider audience.
    • Camera Framing Algorithm to enhance accuracy of frame shots.
    • External Monitor/Projector Display feature for increased versatility.
    • Updated Read Me Instructions for PowerShell compatibility.
  • Planned Changes:
    • Image Carousel for Admins for better "fill image" feature customization.
    • Adjusting Color Segmentations to improve laser tracking accuracy.
    • Testing on Different Canvas Mediums to ensure optimal performance.
